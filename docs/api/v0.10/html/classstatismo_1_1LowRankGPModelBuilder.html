Detailed Description

template<typename T>
class statismo::LowRankGPModelBuilder< T >

A model builder for building statistical models that are specified by an arbitrary Gaussian Process. For details on the theoretical basis for this type of model builder, see the paper

A unified approach to shape model fitting and non-rigid registration Marcel LÃ¼thi, Christoph Jud and Thomas Vetter IN: Proceedings of the 4th International Workshop on Machine Learning in Medical Imaging, LNCS 8184, pp.66-73 Nagoya, Japan, September 2013
